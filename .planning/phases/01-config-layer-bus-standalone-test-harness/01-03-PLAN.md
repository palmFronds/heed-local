    Implement src/bus.js with a module-private `const target = new EventTarget()` that is never exported. Export `publish(type, detail)` that calls `target.dispatchEvent(new CustomEvent(type, { detail }))` — the payload MUST be wrapped in the `{ detail }` options object (a bare `new CustomEvent(type, detail)` silently yields undefined on subscribers, Pitfall 3). Export `subscribe(type, handler)` that wraps the handler as `(e) => handler(e.detail)`, registers it via `target.addEventListener(type, wrapped)`, and returns an unsubscribe closure calling `target.removeEventListener(type, wrapped)`. Do NOT bind to or dispatch on `document` or `window` anywhere (Pitfall 2 — that is a host-page eavesdropping surface). This module is the single place in the codebase that constructs a CustomEvent. No network I/O, no imports.
