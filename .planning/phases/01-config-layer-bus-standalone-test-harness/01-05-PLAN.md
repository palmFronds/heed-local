    A standalone static test harness (test-harness/index.html) that loads dist/sdk.js, exposes all 7 CONTRACT.md data-heed selectors as real elements, validates the bundled demo config at load (hard-fail on invalid), and provides a synthetic-signal debug panel whose four buttons publish signal:detected events onto the private bus, with an on-page log showing receipt via a decoupled subscriber.

    5. Confirm no config error is thrown on load (the bundled demo config validated cleanly). Optionally, to prove the hard-fail path, temporarily nothing is required — this is covered by the CFG-02 automated test.
    6. Click each of the 4 debug-panel buttons (touch_hesitation, blur_incomplete, scroll_reversal, back_intent) and confirm each click appends a corresponding signal:detected entry to the on-page log (proving publish → subscribe receipt), and that the logged payload contains only type/targetSelector/bbox/timestamp (no field values).
  </how-to-verify>
